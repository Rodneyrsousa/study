## Instalando WSL

- Ativar ou desativar recursos do windows 
	- Hyper-V
	- Plataforma de maquina virtual
	- Subsistema do Windows para Linux	

# reinicie a máquina

- Instalar o WSL2
```bash 
wsl --install
# reinicie a máquina
```

- Listando distribuições disponíveis
```bash
wsl --list --online
wsl --install -d <DISTRO_NAME>
# reinicie a máquina
```

- Configure a distribuição
